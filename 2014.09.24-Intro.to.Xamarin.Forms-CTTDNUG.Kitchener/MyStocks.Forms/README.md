MyStocks.Forms
==============

Xamarin.Forms example of querying stock quotes and using text to speech apis.


## Features

* Text To Speech APIs
* Query Yahoo APIs for stock quotes
* MVVM with Databinding to XAML UI
* Custom fonts per platform
